# ArkidilloEngine
Homemade 2d Java based game engine

## Installation
1. Install JDK
2. Place user code in `ArkidilloEngine/src/userCode`

## Usage

### Create + Display Sprite
Create a `Sprite` object, by passing the object its location, an image file, and its `Scene` object.
